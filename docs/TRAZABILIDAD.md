## Finalidad

Este archivo permite registrar los cambios relevantes realizados en el repositorio, con el fin de mantener evidencia de evolución, mejora, actualización y continuidad del desarrollo.

La trazabilidad ayuda a identificar el proceso de construcción del proyecto, las decisiones tomadas y la progresión del ecosistema desde su etapa inicial hasta futuras versiones.

## Cambios que deben registrarse

Se recomienda registrar en este archivo:

- Cambios de estructura.
- Actualización de enlaces principales.
- Incorporación de módulos.
- Cambios en cabecera, footer o navegación.
- Incorporación de nuevos recursos.
- Ajustes visuales relevantes.
- Mejoras de accesibilidad.
- Correcciones técnicas significativas.
- Cambios de dominio, alojamiento o publicación.
- Incorporación de documentos de licencia, actas o inventarios.

## Criterio de control

Todo cambio relevante debe quedar respaldado mediante commits, archivos actualizados o documentos de control. Esto permite conservar una línea histórica clara sobre la evolución del repositorio y su relación con el Ecosistema Virtual Accesible (EVA).

## Nota de preservación

No se recomienda eliminar este archivo. Si el repositorio cambia de estructura, este documento debe mantenerse o trasladarse a una carpeta documental equivalente.
