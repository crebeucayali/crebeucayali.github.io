# Inventario inicial de componentes

Repositorio: Inicio  
Proyecto: Ecosistema Virtual Accesible (EVA)  
Autor del desarrollo original: Psicólogo Gabriel Berrospi  
Fecha de referencia documental: 08 de julio de 2026

## Propósito del inventario

Este documento identifica los componentes generales vinculados al repositorio Inicio y su relación con el Ecosistema Virtual Accesible (EVA). Su finalidad es dejar constancia de los elementos que forman parte de la estructura autoral, funcional y documental del proyecto.

## Componentes del repositorio Inicio

El repositorio puede comprender, según su estado de desarrollo:

- Página principal o archivo de entrada del ecosistema.
- Estructura HTML, CSS y JavaScript asociada a la interfaz.
- Cabecera, secciones, tarjetas, botones y bloques de navegación.
- Enlaces hacia módulos, repositorios y páginas complementarias.
- Textos institucionales, textos de presentación y orientaciones de uso.
- Recursos visuales incorporados o enlazados.
- Criterios de accesibilidad aplicados a la estructura del sitio.
- Archivos de configuración necesarios para publicación en GitHub Pages u otros servicios compatibles.
- Documentación interna en formato Markdown.

## Componentes funcionales del ecosistema vinculados

El repositorio Inicio puede articular el acceso hacia los siguientes módulos o espacios del ecosistema:

- CAP: Capacitaciones CREBE.
- BDA: Banco Digital Accesible.
- MEA: Materiales Educativos Accesibles.
- JIA: Juegos Interactivos Accesibles.
- NI: Notiinclusivo.
- RA: Repositorio Accesible.
- AC: Accesos Complementarios.
- Otros módulos o submódulos futuros que el autor incorpore.

## Componentes autorales

Se consideran componentes autorales del proyecto:

- Concepto general del ecosistema.
- Organización modular.
- Lógica de navegación.
- Distribución de accesos.
- Selección y jerarquía de módulos.
- Identidad funcional de la página de inicio.
- Redacción original de textos de orientación.
- Criterios de presentación visual.
- Estructura de documentación interna.
- Integración progresiva de contenidos educativos y accesibles.

## Componentes institucionales o de terceros

Cuando el repositorio utilice nombres institucionales, logos, imágenes, pictogramas, documentos, videos, enlaces, materiales descargables u otros recursos de terceros, dichos elementos deberán entenderse como componentes diferenciados del desarrollo original del autor.

Su uso debe respetar la titularidad, fuente, licencia o autorización correspondiente.

## Exclusiones

No se consideran incorporados automáticamente a este inventario:

- Futuras versiones del ecosistema.
- Nuevos módulos o repositorios creados después de la fecha de referencia.
- Materiales institucionales posteriores no integrados expresamente.
- Recursos de terceros sin autorización o licencia compatible.
- Plataformas paralelas o proyectos personales futuros del autor.

## Observación

Este inventario tiene finalidad documental, organizativa y probatoria. Puede actualizarse cuando el repositorio cambie de versión, incorpore nuevos archivos o se formalice una entrega específica.
